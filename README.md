# CSE-412-final-project

# How to get started
1. API directory and UI directory both have readmes on how to get started.
2. You will need both Python & npm.
3. In order to run the application, first do "python app.py" in the api directory. This will set up the backend server that communicates with the database.
    - 3.1 Do not click on the link the terminal gives you, there is nothing there. It will just give you a 4** error.
4. For the frontend, run "npm run dev" in the ui directory. Here you can actually click on the link given to you in the terminal and it should be able to communicate with the backend.
